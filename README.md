# 24-Week AI-Aware Cybersecurity Transition Project

This repository documents my self-directed, hands-on 24-week journey into cybersecurity.

I am transitioning into cybersecurity **with no formal training**, so I designed and built my own structured program focused on:

- AI-aware phishing detection  
- GRC governance  
- SOC automation  
- Red Team simulations (safe & synthetic)  
- Purple Team detection improvements  
- Deepfake + voice cloning defensive analysis  

The project models modern social engineering threats inspired by groups like **Scattered Spider**, and turns them into:
- A real detection pipeline  
- Governance controls  
- Operational workflows  
- Automation logic  
- Weekly progress documentation
